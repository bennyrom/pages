¿Qué es GITHUB?

Git es un sistema de control de versiones. Un sistema de control de versiones nos va a servir para trabajar en equipo de una manera mucho más simple y optima cuando estamos desarrollando software. ... Cuando acabamos de desarrollar nuestro código, utilizamos Git para mezclar los cambios con los otros compañeros


¿Qué es y para qué sirve un repositorio?

Los repositorios son sistemas de información que preservan y organizan materiales científicos y académicos yo a la investigación y el aprendizaje, a la vez que garantizan el acceso a la información. La palabra española repositorio procede del latín repositorium, que significa armario o alacena.
